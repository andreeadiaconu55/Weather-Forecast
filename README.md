# Weather Forecast

## Story

People Always Talk About The Weather...

This application should allow you, as a user to enter a city name (and be gently helped by an autocomplete in selecting it), and see the specific weather in that city. 
The UI/UX will blend in an photo from the city with the rest of the application.

## Elements learned while creating this aplication

- Creating GET requests
- Getting data from an API
- Working with JSON
